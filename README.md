# recent

## Usage

```plaintext
This program shows recently modified files/directories

Usage: recent [OPTIONS] [Target Directory]

Arguments:
  [Target Directory]  File search root. Default: current directory

Options:
  -v, --verbose  Verbose output
  -h, --help     Print help
  -V, --version  Print version
```
